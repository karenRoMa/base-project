AlertViewController

class AlertViewController : UIViewController

View Controller with an alert design to display mainly errors received from the server. It can display any message given and have two buttons:

  • cancel button: Dismiss the alert without any action
  • accept button: Dissmiss or not the alert and perform the action in Alert Delegate if is not nil.
